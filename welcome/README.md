# 🌈 Welcome to StreamWave!

## 🎯 Quick Start Ritual

1. **🎪 Raise the Curtain**
   ```bash
   docker-compose up -d
   ```

2. **🎭 Watch the Magic**
   ```bash
   docker-compose logs -f
   ```

3. **🌊 Surf the Streams**
   - Kafka UI: http://localhost:9021
   - Flink Dashboard: http://localhost:18081
   - Superset: http://localhost:8088
